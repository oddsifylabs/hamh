# Ruth Hermes - Web Developer Worker

Oddsify Labs Web Developer. Runs on Railway and polls HAMH for web dev tasks.

## Setup

```bash
cd /home/ruth/ruth-worker
npm install
npm start
```

## Railway Deploy

Add as a separate service in your Railway project, or run alongside HAMH.

## Task Types

| Type | Description |
|---|---|
| `web-development` | Full-stack web development |
| `frontend` | Frontend code |
| `backend` | Backend/API code |
| `ui-ux` | UI/UX design |
| `deployment` | Deploy sites/apps |
| `maintenance` | Updates, fixes, health checks |
