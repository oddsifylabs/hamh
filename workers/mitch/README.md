# Mitch Hermes - Sales & Marketing Worker

Oddsify Labs Sales & Marketing Manager. Runs on Railway and polls HAMH for tasks.

## Setup

```bash
cd /home/mitch/mitch-worker
npm install
npm start
```

## Railway Deploy

Add as a separate service in your Railway project, or run alongside HAMH.

## Task Types

| Type | Description |
|---|---|
| `lead-generation` | Find and qualify leads |
| `sales` | Sales outreach |
| `marketing` | Marketing campaigns |
| `crm` | CRM updates |
